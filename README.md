JSON Schema Builder

A simple React + Ant Design project to build dynamic JSON schemas with nested fields.

Features

- Add, edit, and delete fields
- Support for String, Number, and Nested types
- Recursive nested fields
- Live JSON preview

Tech Stack

- ReactJS
- JavaScript
- Ant Design
